The next area consists of a paragraph and 2 spans.
  • The paragraph is a block element.
    • Block elements consume the entire width of the parent.
    • Blocks stack on top of one another.
  • The spans are each inline elements.
    • Each span is inside of the paragraph.
    • Inline elements consume the exact amount of space needed.
    • Inline elements might wrap from one line to the next.
    • Inline elements don't stack on top of each other.
    • Inline elements are laid out side-by-side.
  • Take notice of the spacing after the span and how the html is a little different in each case. Spacing can be very tricky.
