Special Characters

Q: Since the less-than and greater-than symbols are used for tags in an HTML document, can we show the actual symbols for less-than and for greater than?

A: Yes... < and >

Q: Since the ampersand symbol was used to be able to show the symbols < and > in an HTMl document, can we show the actual symbol for ampersand?

A: Yes... &

Q: If we were writing an HTML tutorial in an HTML document, how would we show the HTML that needs to be typed for various special characters?

A: Consult the list below for examples.

  • Use &lt; in an HTML document so that the browser renders <
  • Use &gt; in an HTML document so that the browser renders >
  • Use &amp; in an HTML document so that the browser renders &
